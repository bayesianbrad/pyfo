The run inference method should be run externally once the class has been created. I.e assume that they have not written there own model. >>> hmc = MCMC(‘HMC’) >>> samples = hmc.run_inference(nsamples=1000, >>> burnin=100, >>> chains=1, >>> warmup= 100, >>> step_size=None, >>> num_steps=None, >>> adapt_step_size=False, >>> trajectory_length = None, >>> dirname = None) It then returns the samples generated from inference. Alternatively you can set a global directory for saving plots and samples generated.

Parameters:
  • type (nsamples) – int descript: Specifies how many samples you would like to generate.
  • burnin – type: int descript: Specifies how many samples you would like to remove.
